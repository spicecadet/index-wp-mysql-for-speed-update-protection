# Index MySQL for Speed Update Protection
This is a MU plugin that prevents WordPress core updates from reverting database optimizations made by Index WP MySQL For Speed. This plugin will run each time WordPress is updated.

Code for this plugin is based on the following solution and additional detail are available at the following URL: https://www.plumislandmedia.net/wordpress/filtering-database-changes-during-wordpress-updates/ 

## Install Instructions
Move the index-wp-mysql-for-speed-update-protection.php file to your wp-content/mu-plugins directory.
